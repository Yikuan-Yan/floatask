# Changelog

All notable changes to Floatask will be documented in this file.

## [1.0.0] - 2026-03-30

### Added
- Collapsed / expanded dual-view with smooth toggle
- Draggable and resizable frameless window
- Always-on-top pin button
- System tray minimize (X button hides to tray, click tray icon to restore)
- Custom statuses (single-select) and tags (multi-select)
- Smart date tracking: created, started, completed dates auto-set per status change
- Archive system: bulk archive done & not applicable tasks, browse by week, restore
- Markdown support in notes: **bold**, `code`, [links](url), bullet lists
- Full-text search across active and archived tasks
- 3 themes: Ocean Blue, Slate Dark, Coral
- JSON import / export for backup and migration
- Keyboard shortcut: Ctrl+Shift+T to toggle panel
- Settings: auto-start on boot, default new task status, collapsed task count
- Drag-and-drop reorder with insert (not swap) behavior
- Collapsed view: scalable font size, overflow indicator (···) for 3+ active tasks
- Position memory: restores pre-expand position when collapsing
